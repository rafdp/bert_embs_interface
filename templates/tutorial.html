Model repo
Web interface repo
To set up the model you will need python(>=3.6), tensorflow (1.14), pandas, numpy, bert.tokenization, keras, tensorflow_hub.
encode.py is in charge of filtering the Kaggle dataset and tokenizing following the BERT standard.
model_pipeline.py builds the model, trains and validates for 1 epoch.
hparams_full.json is an example of how you can set up all the parameters encode and model_pipeline might need.
predict.py takes comment.txt and predicts over it, outputs a comment_result.npy file.
To set up the web interface you need Flask and flask-wtf.
